# briefings

Investor-facing HTML briefings, hosted on GitHub Pages.

Live at: [https://charles-solanki.github.io/briefings/]

## Adding a new briefing

1. Create a new folder, e.g. `q3-update/`, and place the briefing HTML inside as `index.html`.
2. Add a link to it on the root `index.html` landing page.
3. Commit and push to `main`. GitHub Pages picks it up automatically.

Each briefing is reachable at `https://charles-lutra.github.io/briefings/<slug>/`.
